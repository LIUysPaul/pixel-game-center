# 像素太空射击 — 产品需求文档

## 1. 产品概述

一款像素风纵向卷轴弹幕射击游戏。玩家操控一架像素战机，在星空背景下迎击蜂拥而来的敌机，躲避弹幕，收集武器升级，挑战关底BOSS。

- **核心体验**：高速射击的爽快感 + 弹幕躲避的紧张感 + 升级变强的成长感
- **目标用户**：喜爱街机射击、弹幕游戏的玩家

## 2. 核心功能

| 页面 | 模块 | 功能 |
|------|------|------|
| 主菜单 | 标题/开始/操作说明 | 星空背景+像素标题 |
| 游戏画面 | 战机操控/射击/敌机/弹幕/道具 | 核心玩法 |
| HUD | 分数/生命/武器等级/波次 | 状态显示 |
| 结束界面 | 结算/重开 | 分数排名 |

## 3. 游戏玩法

### 3.1 玩家属性

| 属性 | 初始值 | 说明 |
|------|--------|------|
| 生命 | 3 | 被击中扣1 |
| 武器等级 | 1 | 最高5级，升级增加弹道 |
| 移动速度 | 300px/s | 流畅八方向移动 |
| 射击间隔 | 0.15s | 自动连射 |

### 3.2 操作

| 操作 | 按键 |
|------|------|
| 移动 | WASD / 方向键 |
| 射击 | 自动（持续发射） |
| 炸弹 | 空格（清屏，限量） |

### 3.3 敌人类型

| 敌人 | 血量 | 行为 | 弹幕 |
|------|------|------|------|
| 小型机 | 1 | 直线下降 | 无 |
| 扫射机 | 3 | S形移动 | 2连发 |
| 重型机 | 6 | 缓慢下降 | 扇形5弹 |
| BOSS | 100 | 固定位置 | 多种弹幕模式 |

### 3.4 道具

| 道具 | 效果 | 颜色 |
|------|------|------|
| 武器升级 | 弹道+1（最高5） | 黄色 |
| 生命恢复 | 生命+1 | 红色 |
| 炸弹补给 | 炸弹+1 | 蓝色 |

### 3.5 武器系统

| 等级 | 弹道模式 |
|------|----------|
| 1 | 单发直射 |
| 2 | 双发平行 |
| 3 | 三发扇形 |
| 4 | 三发+侧射 |
| 5 | 五发扇形+侧射 |

## 4. 设计风格

- **主题**：赛博朋克太空风
- **主色调**：深空黑(#0a0015) + 霓虹青(#00ffff) + 霓虹粉(#ff00aa) + 金色道具(#ffd700)
- **画布**：540x720（纵向）
- **特效**：引擎尾焰、爆炸粒子、弹幕发光、屏幕震动
